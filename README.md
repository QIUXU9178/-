
# Dry Bean Dataset 干豆多分类机器学习项目

课程：机器学习与项目实践2026 期末作业
数据集：Dry_Bean_Dataset_Dirty（含缺失、异常、重复脏数据）

## 1. 数据集介绍

数据集为干豆形态特征多分类数据集，自带train/val/test三份脏数据，存在缺失值、重复样本、极端异常值，共多类豆类标签。
原始特征为几何尺寸特征，项目完成完整清洗与特征工程。

## 2. 数据处理方案

1. 清洗：删除重复样本、中位数填充数值缺失、众数填充标签缺失、3σ剔除异常值
2. 特征工程：低方差过滤、标准化、PCA降维保留95%信息
3. 鲁棒性实验：添加0.1/0.3/0.5高斯噪声测试模型稳定性

## 3. 实现算法（3种，含课外拓展模型）

1. LogisticRegression 逻辑回归（课堂基础）
2. RandomForest 随机森林（课堂基础）
3. XGBoost 极端梯度提升树（课外自学拓展模型）

## 4. 模型精度汇总表

运行代码后自动生成 output/model_compare.csv，包含训练/验证/测试精度、过拟合差值、推理速度、噪声鲁棒精度。

## 5. 项目运行方式

1. 安装依赖：`pip install -r requirements.txt`
2. 执行全流程：`python src/run.py`
3. 输出文件：图表、指标表格自动保存在output文件夹

## 6. 项目模块划分

- data_loader.py：数据读取、EDA可视化分析
- preprocess.py：脏数据清洗、全套特征工程
- models.py：多分类模型封装
- train_eval.py：模型评估、对比绘图、噪声测试
- run.py：统一入口，一键执行全部流程
